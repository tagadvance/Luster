# Luster

## Installation

Maven

```xml

<dependency>
  <groupId>com.tagadvance</groupId>
  <artifactId>luster</artifactId>
  <version>1.0.0</version>
</dependency>
```

Gradle

```groovy
implementation group: 'com.tagadvance', name: 'luster', version: '1.0.0'
```

Gradle Kotlin

```kotlin
implementation("com.tagadvance:luster:1.0.0")
```
